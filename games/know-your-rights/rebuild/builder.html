All scenes

Loads scenes live from kyr-content-db via the v2 Worker API. Each scene is one JSON object (GET/PUT /api/kyr/v2/scenes/{slug}) — saving writes the whole scene at once. Active controls whether the game serves it.
Flow is driven by each answer's goto, not list order — list order is just authoring convenience. The start card (set in Scene Settings) is where play begins; end cards finish the scene.

Meters

Numeric, hidden in-game. Exactly one is primary — that's the end-of-scene detainment roll, floored at the scene floor. Grade presets are the answer shortcuts.

Flags

Named scene state with a starting value (e.g. door=closed, warrantShown=false). Answers change flags; rules read them. Values are text; true/false are stored as booleans.
